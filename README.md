# FreakFest 2025 Landing Site

This repository contains the FreakFest 2025 landing website (React + Vite + Express + TypeScript).

- Client root: `client/`
- Server root: `server/`
- Shared types: `shared/`

To run locally:

```bash
npm install
npm run dev
```

To build and run production:

```bash
npm run build
npm start
```
